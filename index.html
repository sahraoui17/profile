  • 5 Jan 2022 (Trust Framework): The code for IoT trust management framework (Blockchain-Trust) is available on GitHub
    IoT devices' trust ratings and reputation scores are stored in persistent Blockchain. The project contains several classes that can run on top of iFogSim simulator to simulate trust management in IoT enviroment.
